\##Bash system toolbox



A menu-driven utility script with system tools



\## Features



\- Show date

\--Current folder

\--List folder

\--Disk usage

\--Memory usage

\--Running process

\--Current user

\--Logging actions



\# Run



\--chmod +x sys-tool.sh

\--./sys-tool.sh
